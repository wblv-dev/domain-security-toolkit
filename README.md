# Domain Security Toolkit

Open source Python tool for automated domain security auditing. 35+ checks across TLS, email authentication, HTTP headers, DNSSEC, and OSINT.

**[View the project →](https://wellbelove.org/domain-security-toolkit/)**

## Install

```bash
pip install git+https://github.com/wblv-dev/domain-security-toolkit
domain-audit --domains example.com
```
