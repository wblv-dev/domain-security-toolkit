# Domain Security Toolkit

I noticed a bit of a gap in my tooling for doing domain lookups and research while on a Cloudflare project at work, pulled together this python tool from a bunch of scripts on my work laptop. You can use it for automated domain security auditing, does a bunch of checks and collates the information into a PDF.
